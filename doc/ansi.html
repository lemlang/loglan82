Class ANSI
Class ANSI implements simple operations in the text mode.
  • unit GotoXY:procedure(x,y:integer)
    Moves the text cursor to the position (x,y)
  • unit SetColor:procedure(color:integer)
    Sets the foreground color for text writing.
  • unit SetBackground:procedure(color:integer)
    Sets the background color for text writing.
  • unit Bold:procedure
    Sets the font to bold font.
  • unit Normal:procedure
    Sets the font to normal font.
  • unit Clear:procedure
    Clears terminal.
